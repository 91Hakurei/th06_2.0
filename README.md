# th06_1.5
东方project自制小demo，殴打蕾咪

<Br/>【基本操作】
<Br/> 方向键移动
<Br/>shift减速聚焦火力
<Br/>Z攻击
<Br/>X大招（二妹）

  【内容】
  有完整的游戏流程
  敌人会掉落各种奖励，拾取可以获得对应的加成
  有BGM，可以流程的切换
  有终符，坚持20秒

  2020-5-13
  添加了暂停功能 ESC，可以方便截图弹幕
  2020-5-18
  添加了手动无敌功能，space，可以方便观赏弹幕
